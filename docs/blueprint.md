# **App Name**: Smart Study MPSE

## Core Features:

- Admin-Secured PDF Management: Secure admin login via secret code unlocks dashboard to add, manage PDFs by paper/subject.
- Dynamic Subject/Paper Creation: Admins can dynamically create and manage papers/subjects that instantly reflect on the home screen.
- Free/Paid PDF Logic: Users encounter locked icons for paid PDFs, prompting purchase. Free PDFs are directly accessible.
- Revenue Analytics Dashboard: The admin dashboard provides visual analytics (charts/graphs) for revenue, PDFs, users.
- Admin Role Selection & Verification: Restricts admin panel access to only validated accounts via code phrase to maintain content security.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to invoke trust, stability, and intelligence, appropriate for education apps.
- Background color: Light blue (#E0F7FA), subtly desaturated, creating a calm and professional backdrop.
- Accent color: Teal (#26A69A), slightly different in brightness and saturation, for CTAs.
- Headline font: 'Poppins', a geometric sans-serif. Body font: 'Inter', a grotesque-style sans-serif. The former is for the 'Smart Study MPSE' title and related larger text, and the latter is for descriptive material and user interface.
- Use flat vector icons with subtle animations for menu items.
- Full-screen layouts with modern glassmorphism-style form boxes and clearly defined sections.
- Smooth transitions, button press effects, and subtle animations for opening cards/menus.
